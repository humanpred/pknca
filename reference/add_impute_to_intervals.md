# Add the imputation column to the intervals, if it is not already there

Add the imputation column to the intervals, if it is not already there

## Usage

``` r
add_impute_to_intervals(object)
```

## Arguments

- object:

  The PKNCAdata object to impute data within

## Value

The PKNCAdata object with an impute column added to the intervals (if it
is not already there) and the object\$impute set to that column name
