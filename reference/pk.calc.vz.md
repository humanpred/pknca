# Calculate the terminal volume of distribution (Vz)

Calculate the terminal volume of distribution (Vz)

## Usage

``` r
pk.calc.vz(cl, lambda.z)
```

## Arguments

- cl:

  the clearance (or apparent observed clearance)

- lambda.z:

  The elimination rate (in units of inverse time) for extrapolation

## Details

vz is `cl/lambda.z`.
