# Calculate the effective half-life

Calculate the effective half-life

## Usage

``` r
pk.calc.thalf.eff(mrt)
```

## Arguments

- mrt:

  the mean residence time to infinity

## Value

the numeric value of the effective half-life

## Details

thalf.eff is `log(2)*mrt`.
