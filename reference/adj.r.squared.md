# Calculate the adjusted r-squared value

Calculate the adjusted r-squared value

## Usage

``` r
adj.r.squared(r.sq, n)
```

## Arguments

- r.sq:

  The r-squared value

- n:

  The number of points

## Value

The numeric adjusted r-squared value
