# Set or get a sparse_pk object attribute

Set or get a sparse_pk object attribute

## Usage

``` r
sparse_pk_attribute(sparse_pk, ...)
```

## Arguments

- sparse_pk:

  A sparse_pk object from
  [`as_sparse_pk()`](http://humanpred.github.io/pknca/reference/as_sparse_pk.md)

- ...:

  Either a character string (to get that value) or a named vector the
  same length as `sparse_pk` to set the value.

## Value

Either the attribute value or an updated `sparse_pk` object
