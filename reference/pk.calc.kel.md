# Calculate the elimination rate (Kel)

Calculate the elimination rate (Kel)

## Usage

``` r
pk.calc.kel(mrt)
```

## Arguments

- mrt:

  the mean residence time

- kel:

  is `1/mrt`, not to be confused with lambda.z.

## Value

the numeric value of the elimination rate
