# Assert that a value is a valid AUC method

Assert that a value is a valid AUC method

## Usage

``` r
assert_aucmethod(method = c("lin up/log down", "linear", "lin-log"))
```

## Arguments

- method:

  The method for integration (one of 'lin up/log down', 'lin-log', or
  'linear')

## Value

`method` or an informative error
